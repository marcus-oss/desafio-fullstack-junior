# Sistema de Api Voluntário 

Sistema de gerenciamento de voluntários desenvolvido com React + Vite + TypeScript.

## Tecnologias

- React
- TypeScript
- Vite
- React Query
- Axios
- TailwindCSS
- React Hook Form
- Zod

## Funcionalidades

- Cadastro de voluntários
- Edição
- Inativação
- Filtros
- Feedbacks
- Validação
## Configuração da API

Crie um arquivo `.env`:

```env
```
Certifique-se de que a API FastAPI esteja rodando localmente.
